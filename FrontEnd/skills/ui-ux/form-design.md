# Enterprise Form Design Skill

Design premium enterprise forms.

Principles:
- Reduce friction
- Increase readability
- Clear validation
- Smart grouping

Rules:
- Use section grouping
- Add helper text
- Clear required indicators
- Inline validation
- Auto-save where possible

Input Style:
- Large tap targets
- Smooth focus states
- Subtle border glow
- Animated labels

UX:
- Keyboard friendly
- Accessible
- Fast completion
- Progressive disclosure

Advanced Features:
- Multi-step forms
- AI autofill suggestions
- Smart dropdown search
- Dynamic validation
