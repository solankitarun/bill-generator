# Cinematic Animated Background Skill

Create cinematic animated backgrounds for enterprise applications.

Goals:
- Premium feeling
- Smooth GPU animations
- Elegant gradients
- Layered depth
- Ambient movement
- Low distraction

Preferred Effects:
- Aurora gradients
- Floating particles
- Mesh gradients
- Noise textures
- Light rays
- Animated blobs
- Floating grid
- Glow effects
- Parallax layers

Animation Rules:
- Use slow movement
- Avoid aggressive motion
- Maintain readability
- 60fps smoothness
- GPU accelerated transforms only

Preferred Libraries:
- Framer Motion
- Three.js
- React Three Fiber
- OGL
- Lenis
- GSAP

Never:
- Reduce text readability
- Overuse blur
- Use distracting flashes
- Create heavy CPU rendering

Reference Style:
- Apple keynote visuals
- Stripe backgrounds
- Linear landing pages
- MotionSites style
