# Enterprise Dashboard Design Skill

Create modern enterprise dashboards.

Dashboard Rules:
- Strong visual hierarchy
- Data-first design
- Minimal clutter
- Consistent card system
- Readable metrics

Components:
- KPI cards
- Analytics charts
- Data grids
- Filter panels
- Activity timelines
- AI insight panels
- Notification center

Style:
- Large spacing
- Rounded cards
- Soft borders
- Subtle glass effect
- Minimal shadows

Layout:
- 12-column grid
- Responsive widgets
- Sticky navigation
- Adaptive sidebar

Charts:
- Smooth animated charts
- Minimal chart noise
- Interactive tooltips
- Gradient areas

Reference:
- Linear
- Vercel
- Stripe
- Retool
- Supabase
