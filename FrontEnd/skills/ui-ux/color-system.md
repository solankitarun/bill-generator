# Enterprise Color System Skill

Create modern enterprise color systems.

Rules:
- Use semantic colors
- Maintain accessibility
- Use neutral-heavy palettes
- Use limited accent colors

Preferred Style:
- Slate/zinc base
- Indigo accent
- Emerald success
- Rose destructive
- Amber warning

Avoid:
- Oversaturated colors
- Neon overload
- Random gradients

Dark Mode:
- True dark surfaces
- Soft elevated layers
- Accessible contrast
