# Enterprise UX Rules

Always:
- Maintain clean hierarchy
- Use smooth motion
- Prefer subtle animations
- Use accessible contrast
- Use reusable components
- Keep interfaces scalable
- Prioritize readability

Avoid:
- Over-animation
- Cluttered layouts
- Neon overload
- Excessive gradients
- Heavy glass blur
- Inconsistent spacing
