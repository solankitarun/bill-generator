# Motion Animation Skill

Create smooth enterprise-grade animations.

Animation Principles:
- Motion should guide attention
- Motion should communicate hierarchy
- Motion should feel physical
- Motion should improve UX

Preferred Motion:
- Fade up
- Scale fade
- Stagger animation
- Smooth layout transitions
- Shared layout animations
- Magnetic hover
- Scroll reveal
- Smooth accordion motion

Timing:
- Fast UI: 150ms–250ms
- Medium UI: 300ms–450ms
- Hero sections: 600ms–1200ms

Easing:
- easeOut
- anticipate
- cubic-bezier(0.22,1,0.36,1)

Avoid:
- Bounce spam
- Excessive spring physics
- Laggy transforms
- Animation overload

Preferred Stack:
- Framer Motion
- GSAP
- Motion One
- Lenis
