# Cinematic Animation System Skill
## Animation Style
Use:
- smooth motion
- layered transitions
- parallax effects
- glass reflections
- animated gradients
- blur motion

## Libraries
Preferred:
- Framer Motion
- GSAP
- Lenis
- React Spring

## Background Animations
Allowed:
- particle effects
- animated grid
- gradient waves
- floating blur orbs
- subtle mesh gradients

Avoid:
- flashy neon overload
- distracting motion
- laggy effects

## Motion Principles
Animations should feel:
- smooth
- elegant
- expensive
- premium
- cinematic

## Page Transition
Use:
- fade + blur
- slide motion
- scale animation
- stagger children

## Scroll Effects
Allowed:
- reveal on scroll
- parallax
- sticky sections
- depth transitions

## Performance
Always optimize:
- transform instead of top/left
- will-change
- GPU acceleration
- lazy motion loading
