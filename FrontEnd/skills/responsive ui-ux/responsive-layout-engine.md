# Responsive Layout Engine Skill

## Layout Rules

### Containers
Use:
- max-width containers
- fluid layouts
- adaptive spacing

### Grid System
Preferred:
- CSS Grid
- auto-fit
- auto-fill
- minmax()

### Responsive Rules

Desktop:
- multi-column layout

Tablet:
- reduced columns

Mobile:
- single column

### Responsive Components
Components must:
- stack properly
- resize fluidly
- avoid overflow
- adapt spacing

- iPhone SE
- iPhone Pro Max
- Samsung devices
- iPad
- MacBook
- Ultra-wide monitors
