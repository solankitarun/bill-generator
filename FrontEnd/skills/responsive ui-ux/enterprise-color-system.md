# Enterprise Color System Skill

## Goal
Build premium color systems.

## Color Principles

### Use
- neutral base colors
- soft gradients
- controlled accent colors
- accessible contrast

### Avoid
- random bright colors
- excessive saturation
- inconsistent palettes

## Recommended Colors

### Neutral
- zinc
- slate
- gray
- stone

### Premium Accent Colors
- indigo
- violet
- cyan
- emerald
- blue

## Gradient Style
Use subtle gradients.

Example:
- dark blue → violet
- slate → indigo
- black → zinc

## Dark Mode
Dark mode should:
- reduce eye strain
- use layered contrast
- maintain readability
