# Typography and Spacing Skill

## Typography
Use modern fonts:
- Inter
- Satoshi
- General Sans
- Plus Jakarta Sans
- Geist

## Typography Scale

### Heading
- bold
- large
- tight spacing

### Body
- readable
- high contrast
- comfortable line height

## Spacing System
Use 8px spacing system.

Examples:
- 8px
- 16px
- 24px
- 32px
- 48px
- 64px

## Layout Breathing
Enterprise UI requires whitespace.
Never overcrowd screens.
