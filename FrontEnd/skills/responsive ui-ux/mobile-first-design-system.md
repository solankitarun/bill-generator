# Mobile First Design System Skill

## Philosophy
Design for mobile first.
Desktop is an enhancement layer.

## Mobile UI Standards

### Navigation
Use:
- sticky bottom navigation
- floating action button
- collapsible sidebar
- swipe gestures

### Mobile Spacing
Use:
- 16px side padding
- 12px card gap
- 8px micro spacing
- 24px section spacing

### Mobile Typography
Use fluid typography:

Example:
font-size: clamp(14px, 2vw, 18px)

### Mobile Cards
Cards must:
- fit single hand usage
- have large touch area
- avoid tiny buttons

### Mobile Dashboard Rules
On mobile:
- convert tables into cards
- use accordion layout
- collapse filters
- simplify charts

### Touch UX
Ensure:
- minimum 44px touch targets
- proper gesture support
- safe area handling
- keyboard-safe layouts

### Mobile Performance
Avoid:
- heavy blur
- excessive animations
- large bundle size
- unoptimized images
