	# Performance First UI Skill

## Goal
Maintain premium UI without sacrificing performance.

## Performance Rules

### Optimize
- images
- animations
- fonts
- bundle size
- lazy loading

### Use
- code splitting
- dynamic imports
- optimized SVGs
- responsive images

### Avoid
- massive libraries
- layout shift
- animation jank
- unnecessary re-renders

## Mobile Performance
Prioritize low-end Android performance.
