# Enterprise Responsive UI Skill

## Objective
Create enterprise-grade responsive UI that looks premium on:
- Mobile
- Tablet
- iPad
- Laptop
- Desktop
- Ultra-wide screens

## Core Rules

### 1. Mobile First
Always start design from mobile layout first.

Preferred breakpoints:
- 320px
- 375px
- 425px
- 768px
- 1024px
- 1280px
- 1440px
- 1920px

### 2. Responsive Layout
Use:
- CSS Grid
- Flexbox
- clamp()
- minmax()
- fluid typography
- responsive spacing

Never use fixed width layouts.

### 3. Enterprise Visual Style
UI must feel:
- clean
- minimal
- spacious
- modern
- premium
- professional

Avoid:
- overcrowded UI
- large borders
- outdated shadows
- random colors

### 4. Card Design
Use:
- 16px–28px radius
- soft shadow
- layered depth
- hover elevation
- blur backgrounds

### 5. Motion
Use smooth transitions:
- 200ms–400ms
- ease-in-out
- spring animations
- subtle hover effects

### 6. Mobile Optimization
Important:
"premium SaaS enterprise application"
