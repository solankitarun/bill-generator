# Enterprise Chart UI Skill

## Goal
Create enterprise-grade analytics visuals.

## Chart Libraries
Preferred:
- Recharts
- ECharts
- ApexCharts
- D3

## Chart Rules
Charts should:
- animate smoothly
- support tooltips
- support dark mode
- remain responsive

## Mobile Charts
On mobile:
- simplify labels
- reduce clutter
- enable horizontal scrolling

## Enterprise Analytics Feel
Inspired by:
- Stripe Analytics
- Vercel Analytics
- Datadog
- Linear Insights
