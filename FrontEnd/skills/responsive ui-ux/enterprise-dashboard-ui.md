# Enterprise Dashboard UI Skill

## Goal
Create high-end SaaS dashboard interfaces.

## Dashboard Principles

### Layout Structure
Use:
- sidebar navigation
- top command bar
- responsive widgets
- modular cards

### Sidebar
Sidebar should:
- collapse on mobile
- support icons + labels
- include active indicators
- support smooth animation

### Dashboard Cards
Cards should include:
- subtle gradient
- depth layering
- soft shadows
- smooth hover
- analytics feel

### Tables
Enterprise tables must:
- support horizontal scrolling
- responsive collapse
- sticky headers
- row hover states
- pagination
- search

### Charts
Charts should:
- animate smoothly
- use soft colors
- support dark mode
- remain readable on mobile

### Enterprise Feel
Design language should resemble:
- Linear
- Stripe Dashboard
- Vercel
- GitHub
- Framer
