# Premium Form Design Skill

## Goal
Create beautiful enterprise forms.

## Form Principles

### Input Design
Inputs should:
- have 14–18px padding
- soft borders
- animated focus state
- subtle shadows
- rounded corners

### Validation
Use:
- inline validation
- animated success/error states
- helper text
- progressive feedback

### Multi Step Forms
Use:
- progress indicators
- step animation
- autosave
- responsive layout

### Mobile Forms
On mobile:
- large input fields
- sticky submit buttons
- optimized keyboard flow
- proper spacing

### Enterprise Feel
Forms should resemble:
- Stripe
- Typeform
- Linear
- Notion
