# Premium Component Guidelines Skill

## Buttons
Buttons should:
- feel tactile
- have hover motion
- support loading state
- support icon alignment

## Modals
Modals should:
- animate smoothly
- blur background
- support mobile fullscreen

## Alerts
Alerts should:
- use subtle colors
- include icons
- support animations

## Dropdowns
Dropdowns should:
- animate open/close
- support keyboard navigation
- support search

## Tabs
Tabs should:
- animate indicator
- support mobile scroll
- remain responsive
